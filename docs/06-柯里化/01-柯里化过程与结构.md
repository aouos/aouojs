## 概念

把接收多个参数的函数，变成接收一个单一参数的函数，并且返回接收余下参数的函数

## 为什么需要柯里化

1. 函数单一原则

## 柯里化过程

```js
function add(x, y, z) {
  return x + y + z;
}

const cAdd = function (x) {
  return function (y) {
    return function (z) {
      return x + y + z;
    };
  };
};

console.log(add(11, 12, 23));
// 46

console.log(cAdd(11)(12)(23));
// 46
```
